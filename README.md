# maven-repo
Maven repository for my public projects

```
<repository>
  <id>shahart</id>
  <url>https://github.com/shahart/maven-repo/raw/releases/</url>
</repository>
```

on Gradle, add this to the 'repositories' section:

`maven { url "https://github.com/shahart/maven-repo/raw/releases" }`
